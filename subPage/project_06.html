HAPPY PANG PANG

web publishing

HAPPY PANG PANG은 반려동물의 간식을 파는 홈페이지로 다양한 종류의 간식들을 소개 하기 위해 리디자인하게 되었습니다.

HAPPY PANG PANG is a website that sells snacks for pets and has been redesigned to introduce various kinds of snacks.

explanation

target

HAPPY PANG PANG은 반려동물을 위한 간식들을 파는 홈페이지로 가독성이 떨어지게 디자인되어 있습니다.
반려동물을 위해 구매를 희망하는 소비자들은 제품의 안정성과 퀄리티를 기대하며 홈페이지를 확인합니다.

style guide

used skill

홈페이지 상단 화면에 사용된 이미지 변화는 animation을 사용해 배경 이미지를 변화했습니다.
변화하는 배경 이미지와 오른쪽 상품의 디테일 이미지를 동시에 변화할 수 있도록 animation을 사용했습니다.
스크롤을 해도 로고와 최상단에 이루어진 고정된 object들은 position의 값을 fixed로 고정을한 뒤 animation을 이용해 움직임을 추가했습니다.
상품들의 다양한 옵션을 상세페이지를 들어가지 않아도 확인할 수 있도록 :hover을 이용해 상세 페이지를 들어가지 않아도 확인 가능하게 합니다.

problem & solution

- Problem : 상품을 구매하려는 소비자들은 퀄리티를 확인하는 것을 우선시 하지만 작은 이미지와 과한 반짝임 효과로 상태를 확인하기 어렵습니다.
-Solution : 퀄리티를 확인해야하는 제품들은 큰 이미지를 사용할 수 있는 디자인을 진행하였습니다.

- Problem : banner로 사용해 이벤트 상품을 광고하지만 진열이 되어 있는 상품 위로 겹쳐져 있어, 광고를 바로 지우거나 있는지 모르는 경우가 많습니다.
-Solution : 광고로 표시되는 이벤트 상품들은 확실히 확인하도록 따로 이벤트 공간을 디자인했습니다.

- Problem : 홈페이지의 가장 큰 장점인 판매되고 있는 상품들의 성분이 가장 최상으로 제공되어 있다는 점이지만 제대로 홍보가 되어있지 않습니다.
-Solution : 직접 농사를 해 최상의 상품을 제공한다는 것을 홍보하기 위해 따로 파트를 나누어 디자인했습니다.

- Problem : 반려동물의 크기에 따라 제품의 구매양이 달라지는 것을 고려해 많은 옵션이 있지만 확인하기 위해 상세페이지로 들어가야하는 점이 불편합니다.
-Solution : 다양한 옵션이 상세 페이지를 들어가지 않아도 확인을 할 수 있도록 마우스 이벤트로 확인이 가능합니다.
