DHL REDESIGN

web publishing

DHL을 사용하는 전세계 사용자들은 홈페이지를 통해 예약을 진행하기에 많은 사람들이 사용하기 편하도록 리디자인하였습니다.

Users around the world who use DHL make reservations through the homepage, so it has been redesigned to make it easier for many people to use.

explanation

target

DHL 홈페이지를 방문하는 방문자들은 DHL의 택배를 사용하기 위해 예약 서비스 페이지를 많이 사용합니다.
하지만 가장 먼저 접하는 내용은 DHL의 택배사에 대한 이야기로 구성되어 있어서 홈페이지 사용에 불편함이 있습니다.

style guide

used skill

가장 많이 사용하는 택배 신청 서비스창을 position값으로 가장 상단에 위치해 있습니다.

고객 서비스는 다양한 문제를 갖고 방문하는 사용자들을 위해 대표적으로 찾는 이유들을 따로 칸을 만들어 flex로 표현하였습니다.

긴 내용들이 사용되어야 하는 부분은 따로 공지칸을 만들어 제목을 붙여두어 짧은 내용으로 이해 가능하도록 디자인하였습니다.

problem & solution

- Problem : 마우스의 변화가 없어 정적이며 사용가능한지 인지가 어렵습니다.
-Solution : hover를 사용해 디자인 변화를 주어 사용 가능한 부분을 인지하도록 도와줍니다.

- Problem : 사용성이 떨어지는 게시판의 경우 긴 글들이 요구되기 때문에 페이지에 부피를 차지합니다.
-Solution : 최대 길이 한줄로한 표현이 되도록 글을 정리해두고, 게시판의 사용을 원하는 사용자들은 따로 이동할 수 있도록 되어있습니다.
